# Tic Tac Toe Game

A simple implementation of the classic Tic Tac Toe game in Java.

The game alternates between the player and the computer until there is a winner or a tie.


## About the Project

This project provides a console-based implementation of the Tic Tac Toe game. It allows a player to compete against a computer opponent in a classic game of Tic Tac Toe.

## Getting Started

### Prerequisites

To run this project, you need to have Java installed on your machine.
